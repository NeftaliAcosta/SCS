Bienvenido a SCS

(Sistema de Control de Soporte). Es un pequeño sistema a la medida creado para un proyecto estudiantil de la universidad Abierta y a Distancia de México (UNADMEXICO.MX)

Historia

Me encuentro estudianto en la Unadmexico.mx la carrera de IDS (Ingeniería de Desarrollo de Software) y en la materia de Fundamentos de programación nos encargaron desarrollar un programa en C que resuelva una necesidad de nuestro actual empleo. Podíamos elegir resolver una necesidad ficticia o una necesidad real.

Tecnología Utilizada

DevC++ (C) MySql Conctor Instancia Windows Server (AWS Amazon) Mysql Server
